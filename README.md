# Procesamiento de Empleados y Palabras
Este trabajo práctico corresponde a la materia AED1, donde se solicita implementar dos programas completos utilizando estructuras de datos básicas, modularización mediante funciones y procesamiento de información ingresada por el usuario.

El parcial está dividido en dos consignas independientes:

Consigna 1: Gestión de empleados y sueldos

Consigna 2: Procesamiento de listas de palabras

Ambos ejercicios están implementados en archivos separados, siguiendo el formato requerido por la evaluación.

---
## Consigna 1 – Gestión de empleados

✔ Funcionalidades solicitadas

El programa permite:

Ingresar n empleados y sus sueldos.

Guardar los datos en dos listas paralelas.

Calcular el promedio de sueldos.

Obtener el empleado con el sueldo más alto.

Ordenar empleados y sueldos de menor a mayor.

✔ Funciones implementadas

calcular_promedio_sueldos(sueldos)

obtener_empleado_sueldo_max(sueldos, nombres)

ordenar_por_sueldo(nombres, sueldos)

---
## Consigna 2 – Procesamiento de palabras

✔ Funcionalidades solicitadas

El programa permite:

Ingresar n palabras por teclado.

Buscar palabras que contengan una letra dada.

Ordenarlas según su longitud.

Devolver y mostrar los resultados usando funciones dedicadas.

✔ Funciones implementadas

encontrar_palabras_con_letra(palabras, letra)

ordenar_palabras_longitud(palabras)

---
## Tecnologías utilizadas

Python 3.x

Entrada estándar por teclado (input)

Listas, funciones, bucles y ordenamiento
